# Idea: Export FractionalBuddy Tasks to Trello

**Date:** 2026-04-17
**Source:** David — interactive session

## The ask

> "Currently we have tasks in a big list in the tasks tab of fractionalbuddy. Please create a plan to copy or move them to Trello. What is the most efficient way to add tasks to Trello from fractionalbuddy?"

## Why

Clients and some teams David works with already use Trello. Keeping tasks locked inside FractionalBuddy means duplicate entry for those clients. A one-click export to a Trello board puts tasks where the client team already lives, while FractionalBuddy remains the source of truth for David's own execution.
